Proyecto 1: Análisis de sentimientos en redes sociales

En este proyecto, desarrollé un modelo de aprendizaje automático para clasificar los comentarios de los usuarios en Twitter según su polaridad (positiva, negativa o neutra). El modelo se basó en una red neuronal recurrente con una capa de atención. El modelo alcanzó una precisión del 85% en el conjunto de prueba. El código fuente está disponible en [GitHub].

Proyecto 2: Sistema de recomendación de películas

En este proyecto, diseñé e implementé un sistema de recomendación de películas basado en el filtrado colaborativo. El sistema utiliza los datos de las calificaciones de los usuarios de [MovieLens] para generar recomendaciones personalizadas. El sistema se evaluó con la métrica RMSE y obtuvo un valor de 0.95. El código fuente está disponible en [GitHub].
